# 今回取り扱う作品
- ソースコード
    ```tex
    \documentclass{article}
    \usepackage{pgfplots}
    \begin{document}
    \begin{tikzpicture}
    \begin{axis}
        [
            xlabel={\(n\)},
            ylabel={\(t\)},
            % 最小を(0,0)
            enlarge x limits=false,
            width=0.9\textwidth,
            height=0.3\textheight,
            % y軸tick 間隔
            ytick distance=0.5,
            % y軸最小値
            ymin=0,
            % 凡例の大きさ
            legend=inner,
            legend pos=north west,
            % y軸tick 大き
            yticklabel style={font=\small}
        ]
    % グラフ描画A
    \addplot[black, thick] table [x=n, y=av, col sep=comma, mark options={solid}] {data.csv};
    % 凡例登録A
    \addlegendentry{result}
    % グラフ描画B
    \addplot[dashed,domain=0:100000,thin,mark=none,samples={300}] plot(\x, {6 * pow(10,-10) * (\x)^2 + 9 * pow(10,-7) * (\x) - 0.0004});
    % 凡例登録B
    \addlegendentry{\(t=6\cdot 10^{-10}n^2+9\cdot 10^{-7}n-0.0004\)}
    \end{axis}
    \end{tikzpicture}
    \end{document}
    ```
